## 📦 安装

- npm 安装

```bash
npm install @walrus/test --dev --save
```

- yarn 安装

```bash
yarn add @walrus/test --dev
```

## 🔨 使用

package.json 添加如下代码

```
scripts: {
  "test": "walrus-test"
}
```

命令行执行 `yarn test` || `npm run test`
